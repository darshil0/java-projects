# Basic Java Projects

Here is a list of basic Java project ideas with implementations:

1.  **[Calculator](./Calculator/):** A simple calculator that can perform basic arithmetic operations.
2.  **[To-Do List](./ToDoList/):** A console-based to-do list application where users can add, remove, and view tasks.
3.  **[Number Guessing Game](./NumberGuessingGame/):** A game where the user has to guess a randomly generated number.
4.  **[Simple Chat Application](./SimpleChatApplication/):** A simple client-server chat application.
5.  **[Student Management System](./StudentManagementSystem/):** A system to manage student records, including adding, deleting, and updating student information.
